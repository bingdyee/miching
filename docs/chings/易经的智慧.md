# 卦一 乾为天 伟大的天为君的道 
- **卦象**：自然界：天、大河、大平原。六阳并进，纯阳之象，有大的象征。人物：男人、皇帝、君子、总统、首相、祖父。
- **卦意**：勉励人效法“天”的刚健奋发，卦中以“龙”为刚阳的象征。爻位以潜龙—现龙—惕龙—跃龙—飞龙—亢龙的不同阶段为特征，层层推进，展示了阳气萌生，生长，旺盛至穷衰消亡的变化进程，披露事物由初生至物极返终，阳极生阴的哲理。
- **含义**：胸怀大志——刚强进取，自强不息。
- **结构模式**：盛极而衰，大展鸿图，待机而动，惕励奋发，初露头角，潜伏勿动，乾。
- **思维模式**：处于有利条件时，仍须艰苦奋斗，自强不息。道理有三方面：
    - 阶段性：任何事物有萌芽、发展、成熟、衰竭的过程，卦中以港龙、现龙、惕龙、跃龙、飞龙、亢龙作类比。
    - 两点论：任何事物一分为二，没有绝对。有利中含不利，胜利潜伏失败，只有“终日乾乾，夕惕若厉”，才能立于不败之地。
    - 转化性：卦中以阳极转阴，“亢龙有悔”教导人们防止“物极必反”，做事不要走到尽头。
- **哲理启示**：
    - （一）创业初期，要隐忍待机，积蓄力量，而不可轻举妄动。
    - （二）力量不够强大时，要善于借助别人的力量。
    - （三）事业顺利发展，要兢兢业业，小心谨慎，奋斗不已。
    - （四）力量比较强大时，要懂得慎重把握最好的时机。
    - （五）事业顶峰时期，要尽力发挥你所有的潜能。
    - （六）运用刚强、进取最重要的原则：要知进退存亡。
- **决策思维**：当时机还没有到来时，应该隐忍待机。任何事物都有阶段性，不可在不适宜的阶段，无效地表现自己，而应当利用低潮期，先充实自己。像一块蓄电池，先充足电源。但不管任何阶段，都要保持“君子自强不息”的风格。

# 卦二 坤为地 地的包容为臣的遣
- **卦象**：三偶六断，纯阴之象，有大地的象征。
- **卦意**：其意主柔顺，启示人们要有“大地”的承顺万物，有“雌马”之“贞”，卦中以大地、母、柔顺、为特性，意示大地在静止中储存，生育万物，通过卦义，显示直率、方正、宽大、包容承载万物的品性，强调柔顺的处世原则。
- **含义**：为人处事——舒展与柔顺，包容与退让。
- **思维模式**：
    - （1）对不利的条件要有足够的估计，不能掉以轻心，不要盲目乐观。
    - （2）要审时度势，顺应潮流，但必须保持最珍贵最优秀的品质，才能不受损失。
    - （3）要有最后成功的坚定信仰，不要急功近利，不要计较暂时的成败得失。
    - （4）一切言论和行动要特别谨慎。
    - （5）要善于选择可靠的支持力量。
    - （6）保存力量，静观待变。
- **哲理启示**：坤卦象征大地，我们称为大地母亲。她孕育万物，承载万物。因此，启示我们要具有大地的广阔胸襟、浑厚的德行。当天创生万物，地则负载完成孕育生命。我们要具有柔顺遵循的原则，追随而不超越，危害而不排斥。懂得含蓄和谦虚，收敛锋芒，具内方外圆的涵养。
- **决策思维**：要注意处世方式，目前面临的局面，应采取以静观动，以柔制刚。以不变应万变的原则。沉静、包容、柔顺的方式，也能让事业奠定成功的基石。坤卦，阐释地的法则。在宇宙创始万物的过程中，天创生万物，地负载完成生命。地的法则，是安祥与纯正，柔顺的遵循天的法则，是刚毅行动；安静的谨言慎行，所以，但行动方正；追随而不超越，包容而不排斥；具备至柔的性格，这正是为人的基本态度，应当见微知著，了解一切结果，都有累积而成的必然性，必须防患于未然。应当直率、方正、宽大，含蓄而不眩耀，收敛而言行谨慎，谦逊坚持中庸的原则。应当外柔而内刚，外圆而内方。然而，用柔的原则，也不可以极端，极端必然凶险。必须深切体认主从关系，坚持纯正，冷静客观。通权达变，掌握变化，柔而能刚，善用柔的法则，才能够逢凶化吉。

# 卦三 水雷屯 萌芽，充满生的艰难
- **卦象**：象征初生事物，艰难、受挫、困顿、烦恼。
- **卦意**：上坎下震，坎为云，震为雷，雷云相合，刚柔相爻。以初生事物作为象征，如草木萌芽前于地，显示初生、弱小、脆弱。有生命开始盼含义，当生命初期，既有充满希望、光明、活力与生机的一面。又有弱小，经不起摧残的一面，不可轻举妄动，要以静观动，等待时机。
- **含义**：草木初生——艰难时刻。
- **思维模式**：水雷屯，草木从地下冒出的情状，上一阳文即五爻象地，中间象草与根之屈曲，即草初生，欲伸而未能。任何事物在初始阶段，都会遇到艰难。在开始阶段，应细心选择突破口，并要注意：
    - （一）志向纯正，坚守正道。
    - （二）意志坚定，不为一时的困难和反常现象所动摇。
    - （三）明辨取舍，把握时机，勿轻举妄动。
    - （四）进退两难时，宁进勿追。
    - （五）孤立无援时，先求安全，再求发展。
- **哲理启示**：阐释在天地初创的艰难时代，或事业刚起步之时，也是英雄豪杰建功立业的最好时机。我们既要正视创业初期困难重重（屯、坎、赛、困为四大难卦），又要把握这时正是立业建功的良机。
- **决策思维**：万事开头难。像植物破土萌芽承受压力，婴儿出生前出现的阵痛。面对新生事物，新的项目，新的计划要做好一些初创艰难的思想准备。

- **思维模式**：启蒙教育转化的思维模式：人处蒙昧阶段，要合理启蒙教育，既要有严厉措施，更要与人为善，循循善诱，教育转化。重在自然感应，潜移默化。

- **决策思维**：花园里的花草需要栽种、裁剪、修枝。盆景需要修整。人们的成长过程，也需要长辈导师的指引，教育、培训，人们不断“充电”，事业才能不断充实与成功。

# 卦六 天水讼 争论诉讼
- **卦象**：
- **卦意**：上卦乾为天，气清上升；下卦为坎为水则流动下降。升降各自为道，相互产生争辩，有争辩，则要合理解决。引申在事业上，人际关系上关键在如何合理解决矛盾冲突。

# 卦七 地水师 军队战争
- **卦象**：上坤，下坎，地下涌出水，令地面险陷，震动不安。幸得有一物以全力承挡之象。
- **卦意**：上坤，下坎。坤为地，坎为水、为陷、为险。水在地下。地中有水，为聚众推险，扰乱不安。全卦一阳统五阴，一人统众平定局面称为师，即军队。循其变化原则，阐明治军的领导策略。
- **含义**：率兵统将——攻心为上。
- **思维模式**：提出治军或治理企业的组织领导三原则：
    - 重贤能。
    - 严纪律。
    - 重视信息情报。
- **哲理启示**：以统一用兵原则，喻示在竞争与决策中，科学管理与领导的重要性。阐释用兵的原理，强调军队必须是正义之师，方能得到人民的支持。阐明经济竞争、决策等手段必须走正道，指挥必须集中，才能获取成功。
- **决策思维**：为了完成一项任务，需要带领一批人。就像率兵打仗一样，你需要懂得做领导的艺术。知人善任，合理指挥，并且要坚持正道，身先士卒，依靠严密的规章制度、法纪，才能成功。

# 卦八 水地比 相亲依附
- **卦象**：坎水在坤地上流动，地因水而滋润，水凭地而依托，显示互相依承，相辅和合，欢乐之象。
- **卦意**：上坎为水，下坤为地。水向地上滋润，水得地为依附承载。地得水而柔顺，互为依承。象征在一个团体中，群众依附领导形象，领导则为群众谋利。团体间相亲相辅，互相合作，追随领袖，自然吉祥。
- **含义**：交友之道——相亲相辅。
- **思维模式**：人与人之间如何处理相互关系？“物以类聚，人以群分”。但相互间必须诚信，互为辅助，和睦相处为基础，相互合作为吉祥。
- **哲理启示**：人们在社会关系中，互相交往，应有原则：诚信、择善，远小人近君子。自我省身，宽容、无私、包容，才会吉祥。
- **决策思维**：在事业的创建中，合理利用人际关系非常重要，这是事业成功的关键。在诚信的原则下，使朋友之间，上下级之间紧密团结、互相帮助、相互沟通，相互信任、合作，事业一定可成。

# 卦九 风天小畜 小的蓄积小的阻碍
- **卦象**：风行天上，上巽为风，下乾为天，下乾的热气进入上巽之风中，为之畜止的图象。
- **卦意**：上巽，下乾，乾刚巽柔，风行天上为得位，在下乾之气亦受热。欲上升，被上行之风阻止、待用。从而象征君主能以失意潜伏之时，修艺进德以立身命。
- **含义**：积小蓄微——患难亦能修德。
- **思维模式**：卦中以农业耕种收成现象为例：耕种中会出现自然灾害，生产上产生脱节，要及时补耕抢种荒年荒月容易出现“饥荒”，要有一定的积蓄。推广到事业、企业、国家、团体或个人在发展初期，在前进道路上，难免会遇上暂时困境，因此，应该事先有准备（如积蓄力量，原始资产积累等）。
- **哲理启示**：阐述应付暂时困难的原则。任何事物的发展成长过程，均会出现停滞现象，应事先有所预见，另外有所积蓄，为下一步行动做准备。
- **决策思维**：万丈高楼从底起，要想成为元帅、将军，先要当士兵。一切事物的成长都有一个巩固基础的过程。积小富而后成大德，亿万富翁当初也需“原始积累”的过程，每一步成功都需要积聚力量。

# 卦十 天泽履 践履履行
- **卦象**：上乾天，下兑泽，上下尊卑分明有之象。
- **卦意**：上卦为乾为天，下卦为兑为泽，天在上，泽在下，上下尊卑之分为自然之正。故有“礼”之象，礼要人去实行。从而阐述了实践、履行必须柔顺、悦中、坚定志向。以柔制刚，谨慎戒惧。
- **含义**：采取行动——脚踏实地。
- **思维模式**：卦中以“履虎尾”，象征实践履行责任必须谨慎小心，并且要坚守一定的原则，不可刚愎自用，办事要有礼有节。
- **哲理启示**：阐述实践理想，履行办事的原则。人在天地间，能够柔顺和悦，谦卑自处，则无险不可涉。
- **决策思维**：战略决策确定后，需要在许多战术措施去实施。这就是履行办事的原则、如何才能有惊无险，走险路如履平川？这就是要采取脚踏实地，实干慎行，多听不同意见，谦虚宽容吸纳经验，则必能达到目标。

# 卦十一 地天泰 亨通泰平
- **卦象**：上卦坤为地为阴，下卦乾为天的阳，代表乾天的阳气上升，坤地之阴气下降，阴去阳来。大地相交，显现万物生长的亨通、安泰、吉祥之景象。
- **卦意**：上卦为乾为天，下卦为坤为地。乾天之气下降，坤地之气上升，天地两气相交，来往，互成雨泽，滋润育化万物。取这一阴阳协谐的意境，体现亨通、泰平之气象。
- **含义**：国泰民安——没坡无平，无往不复。
- **思维模式**：通过自然界天地交替，阴阳相谐的变化原理，对任何事物的发展变化，均要遵循三点：
    - “无平不坡，无往不复”。事物到一定阶段，要朝反方向转化。
    - 转化需要具备客观条件，内构机制（即内外因素）。
    - “盛极必衰，衰极复生”是规律，人们处事要扬长避短，因势利导。
- **哲理启示**：阐述持盈保泰原则。创业难，守亦不易，不可满足于现状，物极必反，不创新则失旧，要坚持更上一层楼的转级突破创新原则。
- **决策思维**：“天地交而万物通”，“国泰而民安”当自己的人生事业上达到了这种理想的境界时，要考虑持盈保泰，防止物极必反，面临的决策原则是在原有基础上继续创新，一切戒骄傲自满，要制定各项防患措施，采用“内怀刚健，外行柔顺”的策略，使自己立于不败之地。

# 卦十二 天地否 闭寒黑暗
- **卦象**：
- **卦意**：上卦乾为天，下卦坤为地，乾天为刚为阳，坤地为柔为阴。天在上，地在下，阳气本向上，阴气本向下，这样，阳气继续向上，阴气继续向下，各自朝反方向，便不可相交流。不相交，则阴阳之气不能调和，显示闭塞不通，万物不长象。
- **含义**：闭塞不通——突破黑暗，迎接光明。
- **思维模式**：否卦论述事物对立转化原则：
    - 事物必按“否极泰来”阴极返阳，物极必反的规律发展。
    - 事物遭受挫折，闭塞阶段，要看到光明，坚定信心。
    - 在闭塞情况下，要有“包含”、“包容”的豁达心胸。
    - 保存自身实力，促使事态转化。
- **哲理启示**：阐释在局势处于黑暗时期，君子应看到光明，并时刻警惕，防患于未然，这时应保持“有才不露，有德不显，有善不形”的超然姿态。
- **决策思维**：人生不如意事，十常八九。面对挫折，自己应等闲视之，不能乱了方寸。事态闭塞不通，事业得不到发展时，首先要坚守正道，然后疏通“关节”，慢慢改变状态，付出努力后，最终一定可以“否极泰来”，反转局面。

# 卦十三 天火同人 集结和同
- **卦象**：上卦乾天，下卦离火，乾天之气上升，离火示光明，火焰也具有向上燃升，于是二气同道而行，正如蔚蓝的天空与冉冉腾升的旭日。一幅同道相聚而行的图像。
- **卦意**：上乾为天，下离为火，火示光明，向上燃升，如太阳初升，与上乾之天气上升相同，称同人。同道民众聚集。人以类相聚，追求同心合力谋取事业，壮大力量。
- **含义**：与人共事——两人同心，其利断金。
- **思维模式**：人们以类相聚，同道相合，谋求事业时，要注意：
    - 要守正义之道，勿趋邪门。
    - 要弃私念，打破门户之见。
    - 要同心合力共为一个目标奋斗。
- **哲理启示**：阐述同道相聚的原则。启示人们：在社会活动中，最需要人与人之间的和谐协作，人只有凝聚团结，统一和谐，才能战胜困难达到共同目的。
- **决策思维**：古人有“两人同心，其利断金”的说法。当自己在人生事业中，要进入发展阶段时，首先要考虑人的因素，先要寻找志同道合的朋友，或招聘诚实有才能下属，同心协力，为共同的利益而聚集，才能壮大事业，增加财富。

# 卦十四 火天大有 大有收获伟大的事业
- **卦象**：上卦离为光明，下卦乾为天空。呈现出一幅湛蓝的天空中，一轮太阳高挂，万物普照，大有好收获的图像。
- **卦意**：上卦离为火，为光明，下卦乾为天为刚强。象征太阳升在天空之上，万物普照。也表示领导者光明、刚健，足以带领群众，完成伟大的事业。
- **含义**：大有成就——志不可满。
- **思维模式**：“大有”收获与成功，但应有“满而不溢”的涵养：
    - 成功之际，不能得意忘形。
    - 坚持中正、谦虚、谨慎、艰苦奋斗的精神。
    - 继续选贤用能，远小人，近君子。
    - 自强不息，诚信为本，继续创业。
- **哲理启示**：阐述成功后的相应原则：永远保持谦恭、虚心的品德，以及在太平盛世中和谐共处，共同完成未竞事业。
- **决策思维**：当事业上有了大的成就，财富上有了大的收获时，首先要考虑如何与人安顺相处，处处保持谦恭的品质。志不可满，要从更高层次上寻求新的突破。

# 卦十五 地山谦 谦虚的美德
- **卦象**：上卦，坤为地，下卦，艮为山，山在地下之象。一层层的山脉，起形于地上，山本高，却伏于地下，依地而起伏。地与山相接，连绵不断延伸的形象。
- **卦意**：上卦，坤为地，下卦，良为山，即山在地下。山本高，却伏于地下，不自以为高，从而象征人们应有谦逊、虚心的美德。只有谦虚才能获得好结果。
- **含义**：谦虚谨慎——谦受益，满遭损。
- **思维模式**：以山伏于地下，不以自高的自然现象为例，向人们提出“天外有天，山外有山”的忠告。人们在处世时，务必以“谦虚”为重，只有虚怀若谷，“山不厌高，海纳百川”才能成就事业。
- **哲理启示**：地在上而山在下，是高山者而自先吸其高大。从而启示人们：谦虚是发展事业应有的品德，只有“大直若屈，大巧若拙，大智若愚”才是君子风度。
- **决策思维**：“谦受益，满遭损”。目前面临的环境，是虚心地听取周围人的意见，不能刚愎自用，要认真吸取各种合理化建议，这样才有利于事业的进一步发展。

# 卦十六 雷地豫 喜悦安乐
- **卦象**：上卦雷震动在下卦坤的地上。雷动于地，沉静、寂闭的大地，飘出震动之音，象征娱乐，愉悦安逸的景象。
- **卦意**：上卦震为雷，下卦坤为地，雷出地面。坤静，纯阴主闭；闭极则郁结不畅。震，为雷为动，阳动则万物出，故悦，静极思动，闭极始宣，应时顺动为欢。但欢乐恰悦不能过分沉溺，要安居思危，乐不忘忧。
- **含义**：面对安乐——居安思危，乐不忘忧。
- **思维模式**：豫卦是论述思想修养的卦，论述和乐的原则。
    - 提倡正确对待劳逸关系；安不忘忧，仍须奋斗；不可醉生梦死，沉溺逸乐。
    - 利用安乐条件，展开新一轮的“建侯”事业。
    - 不仅要与人共患难，还要与人共欢乐。
    - 居安思危，安居中要有危机感。
- **哲理启示**：阐述和乐原则，告诫人们：要善于利用安乐条件，安乐不忘忧地把取得的安乐机会作为再创下一轮事业的启动条件。
- **决策思维**：在自己的人生奋斗历程中，或在事业上取得成功的阶段，特别要正确处理好劳逸关系。在适当的时候，要适当放松自己，以逸待劳，但却又要牢牢记住，不可过分沉湎酒色，要乐不忘忧。要理解“生于忧患，死于安乐”的辩证关系。

# 卦十七 泽雷随 随从随和
- **卦象**：上卦，兑为泽；下卦，震为雷，表震动的雷潜伏在深渊里，安息不动之状。
- **卦意**：上卦，兑为泽，下卦，震为雷。雷潜伏在泽的深处，有暂时安息之意。因雷随秋季而伏，又随春季而动。从而说明在社会活动中，应选择时机，选择目标，随机而动的重要性。
- **含义**：相随原则——择善而从，借梯上楼。
- **思维模式**：卦中以雷伏深泽，随季节而动的自然现象，论述了人们应如何追求事业，如何随形势发展去进行决策的原则：随机而动，劳逸结合，更要善于以逸待劳。
- **哲理启示**：阐述追随、随和、随机的原则。人与人之间，要舍私利以随和，人与自然之间，要观察天地形势变化而随机，对正义的事业要随时追求。
- **决策思维**：不是人人都能成为最高指挥官，在自己的人生事业生涯中，多是追随式的方式在奋斗。只要自己虚心地随和他人他人也会随和自己，大家相互随和，顺势而为，事业才能成就。但要注意的是随与被随者之间，要注意相互选择，认真协调指挥者与群众的随从关系。

# 卦十八 山风蛊 革新腐败
- **卦象**：
- **卦意**：上卦，艮为山，下卦，巽为风，山下吹起风之象。风入山下，闭息不得振，山阻风不畅通则暗流，则物腐生虫。示意食物腐烂会生虫，事业委靡不振，会败坏而混乱不堪，社会不治，则腐败。
- **含义**：实行整治——挽救然难，振疲起衰。
- **思维模式**：本卦主要讲述革除腐败，引入生机的策略。
    - 任何事物都存在腐朽侵蚀，应有“流水不腐，户枢不蠹”的思维。
    - 要有振疲起衰，挽危于狂澜之斗志。
    - 除腐蚀要彻底，除恶务尽。
    - 把握适宜时机，采取果断措施。
    - 坚持纯正，永拒腐蚀。
- **哲理启示**：阐释振疲起衰的原则：面对腐败，有志君子不可坐以待毙，应有所作为，革新除弊。
- **决策思维**：当自己面对的事物，因其内部出现毛病，滋生腐烂时，需要认真治理，要加强各种防患措施，进行改革，以使其重新焕发生机。在人际关系中，要警惕内部不和、交友不利，要保持高尚气节，整治内乱。

# 卦十九 地泽临 追临临下
- **卦象**：上卦，坤为地；下卦，兑为泽。地在泽上，即地临泽。下泽为兑为悦，而依附上面的地，则上临下象。
- **卦意**：上卦坤为地，下卦兑为悦为泽，地在泽的上面，居高临下。引申为领导者如何对待他统治的民众，强调统治阶层与百姓之间鱼水关系。民众思考无穷，领导者要保护民众于无限。
- **含义**：领导方略——坚持正道，治内安外。
- **思维模式**：阐述领导与被领导之间相互关系的思维模式，领导要讲究艺术：
    - 教育感化与严厉法纪结合，不能强制高压。
    - 领导人要亲自过问事务。
    - 敦厚诚实治民。
    - 善于以智施政。
- **哲理启示**：全卦阐释领导、统治原则。领导者要善于教育启发民众，包容保护民众，建立鱼水之情，办事才能成功。
- **决策思维**：居高临下是领导人物应具备的条件。特别是高层管理者，要站在相对高位，善于用高度的原则，知事之明，知人之能，采用管理方略引导带领部下。并且要深入到下层去，与下属打成一片，把自己的影响力，深入基层去，相临相感，相辅相成。只有这样，自己才能得到众人的拥戴。

# 卦二十 风地观 观看展示
- **卦象**：上卦为巽为风，下卦为坤为地，风行地上，风吹物动，向下展示，引起仰视之情。
- **卦意**：卦巽风为人，下卦坤为地，风本无形可观，触动物器则可观，即展示观察。从而引申以模范的力量引导群众，向人们展示道义和正确的选择方向。
- **含义**：审时度势——正人先正己，抉择进取。
- **思维模式**：阐述人们观察事物的方法、原则。

    - 高瞻远瞩，见识深远。
    - 正人先正己，观察才有目标。
    - 有主见，善体察细致而果断决策。
- **哲理启示**：说明观与瞻的道理，展示道义规范及其正确的选择方向，只有高瞻远瞩，才有深远决策。
- **决策思维**：在自己的生活工作中，不仅需要行动，也需要观察反思，我们要领悟生活工作的深层含义，只有通过旁观事物到内观自身，来检查自己的行为得失，权衡利弊，再经慎密思考，才可辨明是非，以确定进退之策。

# 卦二十一 火雷噬嗑 咬合刑罚
- **卦象**：下卦震雷击发，上卦离火的光明。卦的体象又显示上下颚中间加上一个代表物质的阳文，成为口中咬合咀嚼的形象。
- **卦意**：上卦为离为火为明，下卦震为雷为动。即外显雷霆万钧的威势，内显明察秋毫的严正，以此象征断狱刑罚之道。
- **含义**：治理政事——当断则断，勿受其乱。
- **思维模式**：全卦以震雷之威击发上卦离火之明，从而阐释刑狱问题：
    - 防微杜渐，防止犯罪扩大，小恶必惩，防患于未然。
    - 果断实行刑罚。
    - 完善刑法，怙恶不悛，但要刚柔并济。
- **哲理启示**：通过雷动震发离火寻求光明的卦理。阐释执法的原则及其重大意义。
- **决策思维**：在治理政务过程中，遇到障碍，难以取得成功的时候，只有果断采取行动，正确运用刑罚，铲除不良分子，有法必依，执法必严。善不行不足于扬名，恶不惩不足于铲灭。因此，我们不可固执己见，应避免争讼。争讼不可调解，只有求助法纪，因为公正执法是维持社会的根本。

# 卦二十二 山火贲 装饰
- **卦象**：山下有火，山川秀木有火照明，则显示层层纹理交错在一起的美丽光彩图案。
- **卦意**：上卦，艮为山；下卦，离为火。即山下有火，山生草木，山下有火照明，则草木皆披上一层光环与色彩，显示出纹理交互辉映的文明景象，从而阐述纹饰、礼仪的原则，有礼则安，无理则危。人通过修饰外表，注重外在美，以提高素。
- **含义**：美化形象——有礼可安，无礼则危。
- **思维模式**：贲卦通过叙述文饰、礼仪原则，强调社会与个人在文明礼貌、自我修养方面的重要性。
    - 国家有序，要靠社会文明，个人的修养要有文明、道义、礼仪。
    - 文明修养要贯穿始终，文与质互相配合。
    - 返璞归真，回归自然是文饰的最高境界。
- **哲理启示**：阐释文明礼仪是建立维护秩序，树立个人形象不可缺少的部分，启示人们要重视社会精神文明建设。
- **决策思维**：考虑如何适应客观环境的问题。一个人本质好，又能注重外表修饰，加强文化涵养，表现出高度的文明礼仪，就能美上加美，事事亨通。不论是企业，还是个人，当务之急，是加强精神文明的教育与培训，树立文明美化形象。

# 卦二十三 山地剥 剥落侵蚀
- **卦象**：上卦艮为山，下卦坤为地，山附于地上，本来高耸的山峰随土崩落，山崩土解之势。五阴剥夺一阳，小人专权之象。
- **卦意**：上卦艮为山，下卦坤为地，山附于地。即高山寄托在地上，难保安止，因山受日剥月削，逐渐崩裂之故。意示小人得势，君子困顿时应更慎重，更要隐忍待机，不可冒进之意。
- **含义**：败迹显露——冷静反思，面对现实。
- **思维模式**：卦象显示当小人得势，君子道消，事业困顿时期，应注意的事项：（1）见微知著，及时防范。（2）受打击或处于不利因素时，要正确对待环境。（3）在困顿期，只可忍耐，不可冒进。
- **哲理启示**：任何事物都有穷尽之期，人们应正确对待事物的腐蚀，败落。因此，社会防腐败，个人惧堕落。
- **决策思维**：要顺应社会和自然的发展规律。所谓“月圆自有月缺”时，事物由好转坏的衰蚀过程是一种循序变动的剥落。这时，应把目光投向那些对将来的发展有益处的地方。处理事情不可急躁。要消除不安和忧虑。

# 卦二十四 地雷复 复归复来
- **卦象**：上卦坤为地，下卦震为雷。显示出雷在地下运行的形象，只有阴阳二气相互激发，才产生雷。当阴多阳少时，只有逐步培养阳气，雷这时正安静不动，悄悄地等待恢复元气的时候。
- **卦意**：上卦坤为地，下卦震为雷，为动。雷在地下动。雷动则阳刚之气，恢复上行。从阳气的复元，阐述在事业上从挫折困顿中，重新恢复生气时，应采取的各项对策。
- **含义**：事业复兴——挫折中奋起。
- **思维模式**：卦意显示在一阳初生的恢复期，也就是当事业与人生经历困苦艰难期后，开始恢复、壮大时。应注意事项：
    - 瞄准时机，主动出击。
    - 对准目标，果断积极行动。
    - 发挥主动性，不可坐失良机。
- **哲理启示**：阐释事物恢复、回复的道理，提示人们看准目标，从挫折中奋起。
- **决策思维**：当人生事业经过挫折转入低潮后，慢慢地逐渐恢复时，就像植物经历霜雪后，枝叶又萌芽生长一样。这时要耐心，并保持进取精神，工作上要周密制定好未来的计划、目标，坚持原则，从善人流，一切都会转危为安。

# 卦二十五 天雷无妄 不虚构望外
- **卦象**：上卦乾为大，下卦震为雷，大下响雷，正是阴阳聚合一片阳气勃发，万物鼓动变化的象征，但应沿着正道才能向好的方向发展。
- **卦意**：上卦乾为天，下卦震为雷，天下雷行，阳气勃发，鼓动万物。阴阳相合，按规律创生万物。卦意提醒人们。做事顺应客观规律则可养育万物。
- **含义**：预防谬误——顺其正道，下自成蹊。
- **思维模式**：卦里提示人们面对阳气勃发鼓动的新生事物，应掌握的原则：
    - 要按着规律而行，不可妄为。
    - 顺其自然，脚踏实地。
    - 时机不利，勿妄动。
    - 坚守纯正，面对灾祸要坦然。
    - 乐天知命，以不变应万变。
- **哲理启示**：阐释万事顺应规律，不妄为，不虚伪，沿着正道而行，才能吉祥的道理。只有“人间正道是沧桑”。
- **决策思维**：正确制订符合实际的目标和措施，要安中守本。根据自身的环境，顺势而为，不能有超越现实的非分之想，事事要脚踏实地，不可强求，否则要遭受失败。

# 卦二十六 山天大畜 大蓄积大阴止
- **卦象**：上卦艮，显示停止，下卦乾，显示刚健。全卦组合起来——刚健前进的乾，被艮山所阻止，于是渐停而积聚力量。构成一幅“天在山中”的养精蓄锐，积蓄聚气的图像。
- **卦意**：上卦良为止，下卦乾为健。乾为纯阳，良则阴多阳少，但也属阳卦。阳是大，所以称“大蓄”。乾刚良止，刚健首进，当止则止。由此体现积蓄与停止的两层意义。另外，下卦刚健前进的乾，被上卦艮山阻碍，停下来积聚力量。
- **含义**：积蓄力量——知进退，勿轻举妄动。
- **思维模式**：事业发展到一定的程度，要适当“阻止”——作出调控。以便积存再发展的力量，经济上称“原始积累”。其中要注意几点：（1）发展不能过快，当止则止，适当整休。（2）谨慎勿冒进，适当自我抑制。（3）保持稳健、持重、妥善运用自我力量。（4）注意“积聚”力量。
- **哲理启示**：阐释蓄积与阻止的道理。事业上只有不断积蓄力量，适时调控，才能蓬勃发展。
- **决策思维**：“不积跬步，无以成千里，不积细流，无以成江河”。知识要积聚，财富要积累，要有量的聚集，才有质的变化。当前首要任务是发奋、巩固和充实自己的学识、经验、资财。为下一轮的飞跃积蓄力量。

- **决策思维**：“发奋图强，自力更生”，要靠自己的力量解决困境，自求口食，自养、自给、自足、自律。工作上要接受各种考验，注意发挥自身的潜力。生活上要以正道养身养心，增强德性，节制饮食。由此做下去，一定可以顺畅。

# 卦三十二 雷风恒 恒久恒常

# 卦三十三 天山遁 隐遁退避

- **卦意**：上卦乾为天为健，下卦艮为山为止。天下面有山，面对山坡由健行急步，转为退后停止，当进则进，该退则退，山高不接天，山上可隐。
- **含义**：退避原则——小不忍则乱大谋。
- **思维模式**：遁卦以阴爻向上浸入，小人逐渐强势的卦象，又以天下有高山，有健行的乾和止步的山，暗示君子要知进知退，在不利的形势下，要善于掌握退避、隐藏、解脱的方略。
- **哲理启示**：世间事有进则有退，有显则有隐，适时而出仕，遇乱而退隐。当忍则忍。忍可避邪恶之锋芒，小不忍则乱大谋。
- **决策思维**：“以退为进”。当时机不成熟，准备不充分时，为了避免损失，则需要进行战略撤退。暂时的隐避和退却不仅可以求得安宁，而且可以积蓄力量，以图东山再起。小不忍则乱大谋，在环境险恶的时候，只有退隐，才能洁身自爱，甚至最彻底的“遁”可以超越物外。

# 卦三十六 地火明夷 光明负伤韬晦时期

# 卦三十七 风火家人 家庭伦理
- **卦象**：上卦巽为风，下卦离为火，卦象显示风出自火，是因火蒸热气散发向外形成风。内卦的火是外卦的根源，象征一切事物应于内在为根本，然后才能延伸于外，由卦象示喻人生道路中，治家的重要性。
- **卦意**：上卦巽为风，下卦离为火，火蒸热气向上散发而成风。象征人生事业的起点在于家，由家向外发展。卦意显示家庭是人生的起步点，如何治理家庭，将影响人的事业，人不能治家，则难于治国。
- **含义**：治家原则——讲诚信，有威严。
- **思维模式**：家人卦为人们阐述的是治家的准则，家庭是社会的根本，治理社会应该先从家庭做起，就像由火蒸发散出一样，火是散风的根本。孟子说：“天下之本在国，国之本在家。”所以要先“齐家”才能治国。治家、治国都应遵从一定的原则：基本是树立诚信和威严。即家长诚信，以身作则，处处以诚意、感情感化家人。
- **哲理启示**：风出于火，火旺才风发。一切事物必须以内因为根本，外因是条件。家庭是社会的细胞，细胞活跃，身体才健壮，所以，治家是治国的根本。
- **决策思维**：家庭是社会的一个细胞，家庭是人们在社会事业中奋斗时，停息、休养、依靠的港湾和加油站。家是国之本，治国治家同一理。要有治家原则，既要诚信威严，又要亲情感化。正确处理家庭问题要注意：（1）夫妻之间主持“内政”、“外务”各行正道，坚持操守。（2）家庭“主妇”应保持贤淑、柔顺、谦和。（3）威严、恩爱并行。（4）各尽本分。（5）保持信任，沟通，和谐、安乐的气氛。

# 卦三十八 火泽睽 乖离乖异
- **卦象**：上卦为离火，下卦为泽水，火向上燃烧，水向下流，各自性质背向。离卦为中女，兑卦为少女，二女同居，志不统一。卦象显示了火与水的性质背向，志向相反的情景。
- **卦意**：上卦离为火，火性向上，下卦兑为泽，泽水性向下。象征上下相互背离，而不相交。卦意启发我们在对待事物中，要认清异同。世上之物，千差万别，甚至相互背离，但又隐含丰某些同一性。
- **含义**：相违失和——求大同存小异。
- **思维模式**：卦象通过火向上燃，水向下流；两女同居，志行不一的背了现象。阐述了世上万事万物的矛盾性。喻示人们面对人心各异、矛盾复杂的社会现实时，应遵从：“求大同存小异，同中见异，异中见同”的处事原则。
- **哲理启示**：世界上万事万物都处于矛盾统一体中。如老子所说：“有无相生，能易相成，长往相形，高下相倾、音声相和，前后相随。”以“同中见异，异中取同”达到“求大同存小异”是普遍适用的处世原则。
- **决策思维**：上下之间、领导与被领导之间、夫妇之间不能沟通时，会发生种种误会，这是自然现象。任何事物都可以发现对立组合体相吸相斥的两个方面。在人际关系中，解除矛盾、误会、冲突的方法是沟通。在和谈中，让信任重新建立，但要坚守“求大同存小异”的原则，心诚气和。面临婚姻不和，家人不亲、行人不归时，要主动寻求与沟通、坚持原则，化解纠纷，重新聚合。日常生活中，往往事与愿违，但经过一段时间努力后，又能统一。求同存异，是自然法则。

# 卦三十九 水山蹇 跛脚困难
- **卦象**：上卦以坎水的险，下卦以艮山作止。象征前方有重重险境，只得遇险而止。又以上坎之水流下，以山阻止，即山上有水流下，十分艰难上行之象。
- **卦意**：上卦坎为水，是险；下卦艮为山，是止。象征在前进中遭遇困难险境，停止不行。卦意是人们在各方面遇到困厄时，应冷静地面对并能反身修德，以诚心感动人，从而解除自身所处的困境。
- **含义**：困苦不堪——正确对待困境。
- **思维模式**：以上坎水的险和下艮山的止，示喻遇险而止步，也表示山上有水流下，举步上前，十分困难。从而启发人们，当身处困境时，应有一定策略，根据险境的状态，可以采取：（1）自身力量不足则退而待机。（2）周围有力量，携手而进。（3）无退路，则振奋斗志，置之死地而后生。
- **哲理启示**：人们在与自然与社会斗争中，必然遇到不同类型的险难，在困难面前，要善于利用有利因素，集中智慧解困突围。
- **决策思维**：在前进的路上遇到了险阻，不能抱怨自己。先要认清险境，小心防患，千万不能急躁冒进，进则有险。在这内外交困、进退维谷之际，冷静地自我反省，先在原地不动。慎审寻思各种排除艰险的方略，寻找能出良策智慧的贵人。摸索前行，走出低谷。

# 卦四十 雷水解 解除困难
- **卦象**：上卦震，象征雷，下卦坎，象征水，构成雷雨交加，闪电齐鸣的自然景象。在雷电交集，天空抖擞，倾盆雨下，万物荡涤更新之下，则大自然的一切闭塞沉闷之气全可解除。
- **卦意**：上卦震为雷为动，下卦坎为水为险。喻示采取积极的行动，就可以解除险难，就像自然界雷雨大作可解除闭塞沉闷之气，人们要解除困境，必须采取积极行动。
- **含义**：解除困境——用谋不在勇。
- **思维模式**：以天上雷动，地下降雨，雷雨倾盆解除大地闭塞沉闷之气的自然现象提示人们走出困境，解除危险，要靠自己积极行动，采取有效措施。包括去恶除邪伸张正义，铲除小人等系列行动。
- **哲理启示**：像自然界经常会出现沉闷、闭塞的状态一样，人们也会在人生事业中遇到闭塞的困境，我们要仿效自然界风雷激荡，雷雨降临去荡涤冲洗沉闷闭塞的形态，而积极行动，采取措施。
- **决策思维**：如何迅速合理解除困境？要有明确的计划，并尽快实施。面临种种困境的束缚，往往产生一种快速缓解、挣脱的谷求。其实，“解铃还需系铃人”，因为系铃人最清楚其中的思路、方式。这是启示你明确困境的成因。缓解困境应遵从谋略和柔道。不能鲁莽与硬干。方法是恢复平静，抓住时机，积极行动，一环一环地缓松，要用水一样柔和平易宽阔的方式，化险为夷。

# 卦四十一 山泽损 减少损失
- **卦象**：以自然界山与泽之间的水、土增减为图像，泽水不断蒸发向上，山泥不断下倾，减损了下方的水。所以山高泽低形成损。上卦艮为山，下卦兑为泽。泽深山高，损其深以增其高。
- **卦意**：上卦艮为山，下卦兑为泽。山，土多水少，泽则水多土少，越往上，水越少，象征减少和损失。表示人们要缓解困境，要解决实际问题，必定要先付出一定的代价，这是损。但损与益是互相关联、相辅相成的，要正确对待自我损抑原则。
- **含义**：遭受损失——坚持正道，采取措施。
- **思维模式**：以自然变化中的山高泽低，山上土多水少，泽中水多土少，而它们之间逐渐相互损减增益为例，比喻人生事业的行动中，要正确处理好损减原则，及时抑制自己的欲望。并在自己力量强盛时，自我减损，损己助人，这是合乎自然规律法则的。
- **哲理启示**：自然界有高山与低泽，它们随着时间的推移，不断在水、土之间减损。所谓“损益盈虚，与时偕行”，人们当仿效自然，正确对待减损。
- **决策思维**：“有得必有失，有损才有益。”这是损失与增益之间的辩证关系。在社会生活与社会经济中，一定程度的自我限制或自我损失，完全可以增强自立感和增强获利的基础。面对一些损失，不必惊恐慌张，反而应在事业的规划上进行周密思维，并借助朋友及周围的人际关系的动力，先充实自己的精神。如此，虽然利益上有自损，但将来可能反而在事业上有进展和收获。

# 卦四十二 风雷益 增多受益
- **卦象**：上卦为风、下卦为雷，雷动风起。以风助雷的关系，象征有利。卦又以上卦巽为风为木，下卦的震，表示动，即木在水上漂动，又被风顺吹，而成一帆风顺之势，以此显示顺行而有利益。
- **卦意**：上卦巽是风，下卦震是雷，雷动风发，象征有利。也可将巽作为木与风，震为动，木在水上漂动，又借助风吹之力量，则从顺而行又有助力。象征从顺道而行必然每天都会有增益。
- **含义**：增加收益——助人亦自益。
- **思维模式**：卦象通过雷动风发，风雷相激气势倍增，来阐述利益的关系。也即是促使自我增益，常使自己若有不足，以随时随地增进自己的知识，道德修养和经济能力。
- **哲理启示**：自然界的风与雷相互助长，顺势之下，气势可以倍增，而有益利。人们应效法这一精神，当自身处于强盛时，应当采取损己益人的原则，结果是助人亦可自助。
- **决策思维**：关键在于抓住转瞬即逝的机会以利自己受益。不仅仅在财富上，更重要的是精神方面。特别的知识、智慧、道德上不断要求自我增益，且要经常保持若有不足的求取欲，在受益中要理清助人与求助的关系，不能贪婪和自私。要认真制订出能长期增益自身的计划，重点在知识涵养、道德修养方面。

# 卦四十三 泽天夬 切断决裂
- **卦象**：上卦为兑，兑为悦为缺口，为口舌，下卦为乾，象征健行，强大，即君子以强势驱逐小人。又上卦兑是泽，下卦乾是天，象征泽中的水不断蒸发，在天空中，一旦决注，则把乌云化为雨。比喻消除邪恶的决断力量。
- **卦意**：上卦兑，为缺为口舌，下卦乾为健，象征君子势力强大，仅有少数拨弄口舌的小人，有待驱逐。卦中示喻我们在人生事业进程中，难免有小人及邪恶势力存在，应适时铲除邪恶。伸张正义。
- **含义**：果断抉择——树道德，除邪恶。
- **思维模式**：夬卦以上卦兑中的一阴爻，喻小人和邪恶势力，或称旧势力。其入五阳爻为新生、正义的势力。因而要取果断措施，向邪恶的旧势力宣告决裂，最后取而代之。即我们在外理各项事务时，有利因素占主动地位时，要抓准机会，铲除邪恶，消除不利因素。
- **哲理启示**：如同自然界的变化状态，当泽中的水不断受阳光照射，蒸发成水汽，在天空中变成乌云，这黑压压的云层，只有经过决注，即让它聚集，随之便可化为雨，大地才呈现一片明媚，从而启示人们事事在变化，只有果断地抉择，事业才能成功。
- **决策思维**：当自己周围满是邪恶势力，小人猖狂之时，如何消除邪恶，果断地与小人决断。其原则是：树德务滋，除恶务尽。具体措施是：（1）力量不足时，不可盲动，周密思考度划，谨慎决策。（2）随时戒备，警惕现状，掌握时机。（3）对手强大，则隐忍待机。（4）时机成熟，果断铲熟，果断铲除，不可犹豫不决。（5）坚守正道，除恶务尽。

# 卦四十七 泽水困 穷困

# 卦四十八 水风井 水井用贤

- **思维模式**：卦中通过水桶不断顺人到水里汲水上来，而井水保持洁净，水又不断涌泉补充的现状。阐释人生事业前进中。人们要不断自我省身修养，不断充实自己，更新自己。社会需要蓄才用才，以发展壮大事业。
- **哲理启示**：井水不断地被来往的人汲来饮用，然而井里却能不断涌泉补充，井水洁净依然。由此启示人们，只要自身不断修养，充实更新自己，人与人之间也可相助相养，助人亦自助，这样则事业有成，人间生气盎然。
- **决策思维**：水是生命之源，井是提供源泉的地方。国家的富强需要依靠不断涌现的贤人辅佐治理，个人则需要道德修养并不断开发知识潜能，这是井垣与水的关系。国家、单位要考虑如何创造环境，培养各类人才，清除人事管道的阻塞，让社会贤人创造的智慧像井水一样汲不完、用不尽，知识能量像甘泉保持清澈。

# 卦四十九 泽火革 变革改革
- **卦象**：上卦泽水下卦离火，水火相碰，象征变革。又以水多则泽中之火灭，火旺则泽水蒸干。象征两类不同性质的事物，不断撞击后，产生变化，由此卦象显示变革、改革之势。
- **卦意**：上卦兑是泽，下卦离是火。泽中有火，水多则火熄，火旺则水蒸干。象征一种变革、改革。卦中教导我们通过变革可以不断完善事业。
- **含义**：改革原则——流水不腐，户枢不蠹。
- **思维模式**：革卦通过泽中水火相碰击的变化，阐述了变革现象。变革是宇宙间一切事物变化的必然规律。对每个人来说，实施变革可以自我完善。但要掌握原则：（1）改革是事物发展的巩固。（2）改革要选择适宜时机。（3）要有正确的改革方向。（4）改革要雷厉风行，彻底完成。
- **哲理启示**：如同泽中的火与水相碰击而产生变化，人类社会、自然界，几乎社会中一切事物，都是在改革中前进。
- **决策思维**：社会发展是不断改革，不断进步，但改革要根据自己的实情，采取适当的措施，先改变自己，再影响改变周围。一项重大变革需要成熟的条件，不可轻率从事，要经过反复论证，全盘周密思考后，才能实施。

# 卦五十 火风鼎 食器养贤
- **卦象**：上卦为离火，下卦为巽木，是木入火中。木在火下不断燃烧，呈现出煮食、烹饪之象。古时燃木煮食一般用鼎，卦象构成鼎器之状态。
- **卦意**：上卦离是火，下卦巽是木，用木点燃火，形似燃木煮食物的器具——鼎。以这象征国家组织政权的稳同，并为这稳同的形象去养贤、用人，藏储经济实力。
- **含义**：革新调整——得贤则兴，失贤则败。
- **思维模式**：鼎是古时的煮食具，它不断把煮的食物取出，再放新的食物烹煮，因此，革卦是除旧，鼎卦是取新。所以称“革故鼎新”，也是破旧立新：（1）不破不立，破是为立。破旧才能立新。（2）革新要有贤才。（3）先养贤，后革新。（4）革新用人要慎重。（5）刚柔兼备，革新才可巩固。
- **哲理启示**：古时以鼎烹食，鼎中食物供养圣贤，鼎的形状端庄稳重。启示人们，国家稳固要靠积财养贤。个人要发达，也要不断积聚知识、财富、更新观念和善于用人。
- **决策思维**：人才是社会发展的根本。贤能人物的合理起用形成社会前进的巨大动力。鼎卦启示了稳定与养贤，在稳定环境中，要通过调和包容以吐故纳新。从个人角度上，要不断调节道德规范，纯净思想，合理安排事务。只要你有经验和能力，又能有贤人相助，坚守正道，排除小人，事业上一定成功。

# 卦五十一 震为雷 震劝戒惧
- **卦象**：上卦震，下卦震，一阳生于二阴之下，动行上进，震为雷。上、下卦为雷动，雷声阵阵连绵不断，呈示震动荡的图像。
- **卦意**：上下卦都是震。震是一阳生于二阴之下，动行上进。象征大地的坤，南其下方发生一阳，使大地震动此卦阐述了如何应付动荡，如何面对社会与个人的突发事件。
- **含义**：振威行动——大事临而不乱，治而有威。
- **思维模式**：震卦主要谈自然现象及社会变化中的震动状态事件。它从量变到质变不断地发生。像自然界的地震、雷响。人们也常常要应付自然与社会的突发事件，一般应注意：（1）居安思危，谨慎防乱。（2）防患于未然。（3）临危不惧。（4）适度退让，中正柔顺。（5）借鉴历史同类经验。（6）要积极应付，不可苟且偷安。
- **哲理启示**：以自然界雷声阵阵，天空震荡之势，启示人们应付任何突发事件的原则是以戒慎恐惧之心，进修惠业，自我反省，才可成大事。
- **决策思维**：镇定从容，处变不惊，临大事而不乱。应付突发事件应有这样的稳健风格。当意外的震动或事变降临时，应采取必要的预防措施，但不能恐慌、烦恼和焦虑，在剧变中，我们可能会失去一些东西，但也许可能得到更多机遇，最终可能得多于失。只要坚定信心，以镇静的头脑深思熟虑，自然可以应付自如，并从中获得应付意外事变的能力。

# 卦五十二 艮为山 停止
- **卦象**：艮为山，上下卦都为艮，两座山重叠在一起，表示不可再前行，应该暂时停止。图像显示一种稳重静止之势。艮又表示一阳在二阴之上，阳已到了最上方，也有停止之象。
- **卦意**：上卦艮，下卦艮。艮是一阳在二阴之上，表示阳已到了上方的极点，停止而不能再前进了，象征停止。艮卦阐述了人生发展事业过程中，应掌握适时而止，适当制约的决策原则。
- **含义**：适时制约——见机而行，适可而止。
- **思维模式**：艮卦阐述了事物在一定时期要静止，暂停的道理。物极必反，动极思静，任何事物的运动都要适可而止。但静止要依其几项原则：（1）当止由止，不失其时。（2）从动到止，先在关键部分静止。（3）由动到止，应注意短期的不适应状态。（4）完善利用静止功能。
- **哲理启示**：物极必反，动极思静，阐述事物的静止原则。启示人们在人生事业发展中，要有一定的制约。当动则动，当静则静，动静不失时机则前途必然光明。
- **决策思维**：当行则行，当止则止，旧的慢慢衰亡中止，新的逐渐开始、一动一静，相互为用，事事自我约束，以静待动，内心保持清静，处变不以妄动。处理问题，不超越本职范围。以静观动，可以让思想和行动显得镇定自若，显得有克制，办事有步骤，让人际关系趋于稳定。自己也可以在止步不前时，抓紧学习，修养和反思自身，为以后的前进打下扎实的基础。

# 卦五十三 风山渐 渐进
- **卦象**：上卦巽是木，下卦艮象征山，山上有木，树木渐渐地长高，山的形状也跟着增高，显示出一幅渐渐升起的图像。
- **卦意**：上卦巽为木为顺，下卦艮为山为止。顺从地前行，止于当止，象征渐渐而进。渐卦阐述了人生发展事业中，循序渐进的原则。
- **含义**：循序渐进——千里之行，始于足下。
- **思维模式**：渐卦主要阐释人生事业发展过程中的渐进原则，也是人们不断改变自己，不断适应新环境的问题。归纳如下几点：（1）一切事物随环境的改变而逐渐变化。（2）事物可从不同方向发展而归于统一，殊途同归。（3）事物在逐渐发展中逐步适应新环境，循序渐进可跨越很多障碍。
- **哲理启示**：以自然界的山上树木，逐渐长大，山与木都显出同步渐渐升高的景象，启发人们在人生事业发展进程中，要掌握循序渐进的原则。
- **决策思维**：缓慢发展，循序渐进，宜慢不宜速。就像风遇到山，速度减缓。当事业腾达，容易受阻，应放慢速度，让头脑冷静。生活工作可以执著地努力追求而逐渐成功，但其进程却如鸿雁渐进而后才急行，不乱其秩。万事要知其进退，才能把握成功之道。

# 卦五十四 雷泽归妹 婚嫁
- **卦象**：上震，下兑，震为长男，兑为少女，长男少女之间结合。象征婚嫁，上卦震为雷为动，下卦兑为泽为水。泽中的水随着在不断地震动，象征夫唱妇随的图像。也含有长男少女不相配的错位。
- **卦意**：上卦震为长男，下卦兑为少女。少女与长男结合，象征男女组合婚嫁问题。归妹卦阐释了人生婚姻大事，男婚女嫁的基本原则，其中含有肉欲之恋。
- **含义**：谈婚论嫁——婚姻美满要合乎天地道义。
- **思维模式**：归妹卦以男女婚嫁作比喻，阐述人生寻求归宿问题。其意包含人生谋职，职工寻找单位，官场择选领导等一类问题。卦中的原则是：（1）每人都寻找归宿，应坚守纯正。（2）一次追求不当，可重新追求。（3）追求归宿不理想，以德化之。（4）不中不正，不适于立家归宿，宁肯误期而不违心地投入。
- **哲理启示**：归妹卦通过人生的男婚女嫁的婚姻结合原则，启示我们在人生事业中如何坚持原则。寻求有利于自己发展、壮大、可依托的归宿问题。
- **决策思维**：男婚女嫁是人伦、生活之道。社会万事如同一理，人生需要考虑自己的归宿。这也是人生的正道，必经之路。职员选择企业、上司，或者男女选择异性情侣，如同一种模式，有时是无法任由自己选择的，我们往往为此而苦恼。当自己发现自身处于一种不合适的位置上时，例如工作上。你的才华不被赏识，婚姻上，你的爱人并不是你所爱的人，但你却无能力另谋高处，或另外组合家庭时，只能“求大同存小异”而过日子了。

# 卦五十五 雷火丰 盛大
- **卦象**：上卦震，象征动，下卦离，象征火。火燃烧，发出光明，又不断跃动，展示出一幅盛大的形象。又上卦震为雷，下卦离象征闪电火光。雷声、闪电交集在一起，显出一派盛大的气势。
- **卦意**：上卦震为雷为动的意思，下卦离为火的光明。象征光明的火，燃烧的火为光明。象征光明的火，燃烧的火焰不断跃动，显示盛大、丰盛的意思。卦意表示人生和事业，越是丰盛繁荣，越要秉刚持正。坚守纯正，才能巩固繁荣的局面。
- **含义**：丰盛原则——天地盈虚，与时消息。
- **思维模式**：丰卦借助于天象来比拟人世，阐述了人生和事业盛衰变化的道理。提示人们，丰盛安泰背后也潜伏危机。当事业处于盛大时应注意：目前丰盛，如日月盈亏。盛极必衰，理之自然。盛则易生积忌，应以诚相待。坚持中正，保持清醒，戒骄戒躁，谦虚谨慎。
- **哲理启示**：卦中以“日月盈天”的天象，阐释人生事业、社会变化的盛衰荣枯就像自然界的日、月、天地盈亏，随时间消长。告诫人们，盛时不可骄淫图安逸，才可保持成功的事业长久。
- **决策思维**：“年年岁岁花相似，岁岁年年人不同”。人们喜欢花团锦簇的春天，希望长伴鲜花而不衰，但即使“桃花依旧开”时，“人面却已非”。人们常常希望持盈保泰，让丰盛繁荣永远。但天地盈虚，长盛必衰是客观规律。当自己的人生事业繁荣丰大时，不可骄奢狂妄，适可而止。此时，要考虑多助人，把丰盛的载体扩大，让更多人受益，或支助公益事业，才能让福气延伸。

# 卦五十六 火山旅 旅行不安定
- **卦象**：上卦离火，下卦艮山，象征山上燃起大火，火势一直蔓延开来，不断向前面燃烧着。火前面的动物就要不断远离火源地，就像出外旅行的人，争着赶路的形态一模一样。
- **卦意**：上卦离是火，下卦艮是山。山上燃着火，火势不断蔓延，就像旅行的人，不断争着赶路。旅卦阐述的是在混乱中追求有序，在动荡中找寻安定的原则。
- **含义**：旅行不安定——明智识时，节制不妄。
- **思维模式**：旅卦主要通过旅行这种不安定的环境因素，阐述了人生在动荡不定中如何笃定情绪，约束自身。扩而为之，人生社会出现混乱无序的环境时，应有坦然应对的原则：人生也是一种旅途，不可能一帆风顺。利用智慧，敢冒风险，敢面对坎坷的人生旅途。柔顺地坚守中正，保持安定心态。
- **哲理启示**：火在山上蔓延，驱逐着前面的动物和人群离开，像旅行赶路一般。人生的道路也是一种短暂的旅程，有时坎坷，有时平坦，我们要敢于在动荡中求安定，不断开辟人生的新路。
- **决策思维**：“在家千日好，出门半朝难”。过去人们总是怕出远门。当然，交通便利的今天，旅游也是乐趣。但无论如何，出外不安定，人们忌奔波喜安定。这里决策的焦点是创造安定环境。不论是社会和个人，当务之急是营造安定。在混乱中寻求有序，大处着眼，先求安定。个人方面，不可彷徨苦闷，内心不安定。家庭不可争吵，起居要安定。企业要通过人际关系调节，使工作生产安定，国家要法治加德治，社会要安定。

# 卦五十七 巽为风 进入谦逊
- **卦象**：巽卦以一阴伏于二阳之下，卦象显出伏从与顺入之状，又上下为巽为风，象征风随着风，徐徐而进，无孔不入的状态。
- **卦意**：上下卦都是巽，巽是一阴爻伏在二阳爻的下面，象征伏与顺。卦的意义在于阐述谦让的本质、道理和原则。
- **含义**：谦让原则——海纳百川，有容乃大。
- **思维模式**：巽卦通过一阴爻顺从地伏于二阳之下，风随着风徐徐而入之态。阐述了人们处事待人接物的风格，只有谦逊而入于别人心中，才会为人接纳。因此，在人生和事业中，应保持谦逊原则：谦让不是自卑。优柔寡断不是谦逊。有意的谦逊成虚伪。谦逊应刚健中正。谦逊要适可而止。谦逊要自我抑制。
- **哲理启示**：像大自然的风，徐徐而人，处处体现服从与顺人的态度。启示人们在生活和事业中，必须具备这种谦逊与忍让精神，才能成功。
- **决策思维**：我们把社会上的一切现象，统称为“社会风气”可见风是无隙不入，无处不在。巽的风，卦意启示深入人心的方法与哲理。深入有两种方式：强行与顺从。当你面临不利的环境，你采取与外部冲突、错位的立场，以冲动的、激烈的行为力图改变，说明你没能把握好自己。如果能够顺着事态的发展趋势，不断对其加以修正，深入而又顺从地调整对象，如春风细雨润物无声，把原野吹绿、吹长、吹熟。能以此从容进取，恰如其分，把周围的关系理顺则为上策。

# 卦五十八 兑为泽 喜悦取悦
- **卦象**：上下卦为兑，兑为泽，两个泽合在一起连结着，泽中的水相互流通，滋润万物，从而相悦。又兑卦为一阴前进到二阳之上，如笑口露出来，象征着喜悦之心已表露于外。
- **卦意**：上卦下卦都是兑。兑卦是一阴前进到二阳爻之上，象征欢欣喜悦已表露于外。兑卦为我们阐述的是人在生活和事业中喜悦欢乐的原则。卦的阴爻表示开口，当嘴张开时，既有欢欣畅笑的愉悦，也可能是与人对骂时的口舌。
- **含义**：与人相悦——快乐是人生美酒。
- **思维模式**：以一阴前进到二阳之上，显示笑口已露开，喜悦表于外，从而阐述与人谈吐，使人和悦以及人生的喜悦欢乐原则：彼此沟通，增进和谐，开诚布公，增进信任。自然和悦，不曲意谄媚于人。勿做小人以私利悦人。
- **哲理启示**：卦中以上下两泽合连一起，泽水相互流通，以滋养万物为喜悦。启示人们，欢乐是人生的一杯美酒，但人与人沟通交流的喜悦应光明正大，有诚信，不可奉承谄媚，防小人陷害，而惹口舌争吵。
- **决策思维**：“人生得意须尽欢”，这是唐朝诗人李白留下的名句。在于外，待人接物柔和；在于内，有涵养有实德，人们相处悦，是真诚的欢悦。创造社会引导人生靠智慧感情和诚意。首先，要树立自己的真诚与人相悦，广结朋友，认真调节周围的人际关系，让大家在欢悦和谐中完成工作，融洽感情。

# 卦五十九 风水涣 涣散
- **卦象**：上巽下坎，象征一阵阵的风，在水上吹荡，水面形成一层层、一圈圈的水波，逐渐离涣，像江心的层层涟漪。比喻人们可通过喜悦，使心中的重重闷郁消散。
- **卦意**：上卦巽为风，下卦坎为水。风在水上吹荡，形成层层水波离用开来。象征喜悦使闷郁涣离，卦中阐释人生事业中，要善于处理涣散局面。
- **含义**：分而治之——善于应付涣散局面。
- **思维模式**：涣卦主要是谈如何解除人心涣散的问题，要消除涣散，关键在革除弊政。在处理涣散的原则上，要注意及时采取对策。善于把握周围的力量。适当借助外界力量。要有超前意识。领导者要铲除私心，从自我做起。挽救涣散，应除害兴利。
- **哲理启示**：在自然现象中，风吹荡在水上，水波泛起涟漪一圈圈地离散。启示人们，掌握涣离原理可以让人喜悦，使心中闷郁消散，也可以在领导管理上，把握分而治之的原则，把涣散的人心重归于统。
- **决策思维**：“合久必分，分久必合”。涣散与聚合也是客观事物发展变化的两个极端。这就像水在一定的环境条件下结成冰块，也会随着条件变化融化成液体。在人生事业奋斗中，首先要把积聚在内心的烦恼忧愁化解消散，如冰消雪融，风吹波散。另外，在社会事业上，却要创造环境条件，增进凝聚力，挽回涣散的人心，达成团结、统一的局面。

# 卦六十 水泽节 节制节约
- **卦象**：上坎下泽，以坎中的水不断向下流入泽中，泽中的水装满了，便会溢出来。如何才能使泽中的水不溢出来？一幅节制水源的图像。
- **卦意**：上卦坎为水，下卦兑为泽。水流入泽中，过多过度流入，水会溢出来。象征事物的一种节制。节卦为我们阐述适可而止，既不过度，也不落后的节制原则。
- **含义**：节制约束——治理不知足的弊端。
- **思维模式**：适度的合理节制，是人们生活与事业中所必须的立身处世原则。如何才能合理掌握节制原则？节制要适时知机，加强修养才能自我节制。节制要适可而止，恰如其分。节制要掌握随机应变。
- **哲理启示**：以自然界的坎水不断流入泽中，如何使水在泽中不溢出，不浪费掉为例，从而阐释节制的道理。启示人们在自己的生活事业中应坚持节制的原则。
- **决策思维**：适度节制自我，恰如其分，知足常乐，精神空间无限，肉体生命短暂，生活欲望要适可而止。面对经济事业的发展壮大，要循序渐进，不可过分追求高指标。要保持节制，有计划地扩大。对人对己不可要求过高，当行则行，应止则止，顺其自然。工作沉闷易出错误，生活放荡，常惹祸端。相交朋友，不可超越界限，事事适可而止。

# 卦六十一 风泽中孚 心中诚信智慧通解巽上
- **卦象**：上巽、下兑，巽象征风，兑象征泽。风在泽上不停地吹，其动感撼于水中。水受感动也能虚心接受风，象征诚信。另一象征是中孚的“孚”以“孵”意，即母鸡或母鸟以诚信之心用其体温把小鸡小鸡孵了出来，也是诚信之意。
- **卦意**：上卦巽的风，下卦兑的泽，风在泽上吹而动感撼于水中，象征诚信。诚信是人生处世的立身之本，中孚卦主要阐述了人生事业中的诚信原则。
- **含义**：心诚意恳——忠信是立身之本。
- **思维模式**：通过风吹泽上，波动感于水的景象，提出人生事业中，立身处世的根本是诚信问题。建立减信的原则有几点：（1）慎重地对待。（2）诚信要感应，感应须发自内心。（3）忌犹豫不定。（4）应恰当选择。（5）诚信是相互受益的。（6）不可过度。
- **哲理启示**：由风吹泽上，波动感于水的景象，以及母鸡孵蛋的形态，提出诚信的可贵，启示人们只有恪守诚信的原则，才能使事事与人生达到更高的境界。
- **决策思维**：推心置腹，以诚相待，自有好结果。就像母鸡孵蛋，诚心遮盖，以体温恒守，自能让小鸡化成。思想和行动产生于诚诚实实地对待周围的人，他们会自愿与你合作。有信义的人，人人都愿意听你指挥心里充实。这样，不仅生活愉快，事业也易成功。

# 卦六十二 雷山小过 小过的过度
- **卦象**：上震下艮。震为雷为动，艮为山为止，象征雷在山上响过后，声音渐减小，即为小的过度。又卦辞“飞鸟遗之音”如小鸟叫过后的声音。小过其形如现代的飞机，上下四阴为机翼，中间二阳为机身，有飞机穿云层山峡之象。
- **卦意**：上卦震为雷，下卦艮为山，雷声在山上响，响声过后逐渐减小，象征小的过度。卦中提示人们在人生事业中如何妥善处理问题。掌握过分、过度与适时收敛，即矫枉过正等一类问题。
- **含义**：超过常规——刚柔相济，动静有度。
- **思维模式**：小过卦。通过雷声响山谷，声音渐小飞鸟在山上鸣叫后的回音，向我们展示了事物的过度阶段的处理原则：（1）正确把握过与敛，不可好高鹜远。（2）中正原则下，无妨稍为过度。（3）适当掌握矫枉过正。（4）不可固执，应随时应变收敛。（5）处事太过分，不足成大事。（6）极端的过度有凶险。
- **哲理启示**：雷声在山上响过，声音逐渐变小，又像小鸟在山上，鸣叫的余音会逐渐消失。这就是过度阶段的收敛之道。启示人们在人生事业上应正确遵循过度时期的处世原则。
- **决策思维**：事事要预防过失：过于自信，主观判断有过失；操之过急，行为上有过失；轻举妄动，难于避免损失。生活上，工作事业上，产生过度的举动，要适当收敛，事事要量力而，矫枉过正，不为过。认真克服反应过激和行为过度的倾向。要面对现实。考虑自身能力和条件的局限，不过分去做超能力、超负荷范围的风险事。只要小心谨慎，为自己确立一个基本行为准则，可以少犯或不犯过错。

# 卦六十三 水火既济 完成
- **卦象**：上坎、下离。坎是水，离为火，上面装着水，下面有火燃烧，火燃水沸，象征烹饪已经完成，卦象显示成功的形态。
- **卦意**：上卦坎的水，下卦离为火。上有水，下有火，象征烹饪已经完成，既济卦为我们阐释的是，当我们事业成功、名成功就之后的应变原则。
- **含义**：成功以后——物极必反，盛极转衰。
- **思维模式**：既济卦通过水在上，火在下，火燃水沸。理顺成章表示成功，从而阐述我们在事业成功、达到目的之后的应变原则：成功后，更须审慎而行。坚守中正，不可失志放肆，遭致衰败。戒骄戒躁，保持警惕。要有危机感。狂妄自信，加速灭亡。
- **哲理启示**：人们烹饪食物，锅里有水，炉下有火，火燃水沸，则可完成。但任由下面的火燃烧，水在上方一旦沸而溢出，会把火熄灭，要重新再起炉灶。启示人们：世界上的万事万物既有主观与客观的适应问题，又都遵从物极必反、盛极转衰的规律而行。
- **决策思维**：不可被胜利冲昏头脑。当人生事业取得一定的成果，决不能过于自信。因为成功之后，更容易出现新问题。成功仅仅是暂时的平衡和相对的完成，过后又会有新的矛盾出现。要有“居安思危”的准备，生活中没有永恒不变的东西，成功面前，依然潜伏危机，需要相应采取各项措施，防患于未然。

# 卦六十四 火水未济 未完成
- **卦象**：上离下坎，水火反错。离为火，火性向上燃烧，坎为水，水性向下奔流，卦图显出水与火，背道而行，相互分离之状。
- **卦意**：上卦离为火，火性向上，下卦坎为水，水性向下，水火背道而行，象征未完成。这卦为我们阐述了既济与未济之间的正反合原理。在成功的时候，应有新追求，今天的成功意味着明天会面临新的艰难征程。
- **含义**：事无止境——永不放弃追求。
- **思维模式**：未济卦阐述的是事物成功之后，又要走向反面，相聚之后又要走向相离。因而，人生事业永无止境。未济是新一端事业起步之际，我们要注意的原则：脚踏实地，不可轻率冒进。明辨机理，量力而行。敢于面对险阻。坚守正道，发奋而作。自身磊落，诚信用人。充满信心。
- **哲理启示**：自然界的万事万物都遵循正反面的辨证原理在发展。正如：北方的极端是南方的起点，白天的尽头是黑夜的起点，既济是未济的开端。卦里启示我们：在人生事业中。成功使人永不放弃自己的追求。今天成功，意味着明天有新的艰难要起步。笔者也提醒人生事业处在未济阶段的人们：勇敢起步吧！冬天来了，春天还会远吗？！
- **决策思维**：事物的发展永无止境，我们永远不可放弃追求。今日的成功意味着明日新的征程起步，又是一个重新开始。特别要注意。当成功还未最后确定，尚有遗留问题，有待最后解决时，必须避免过早的夸耀和庆祝。许多人失败在最后的一步。事业有时遇到困难，要决心克服，既要考虑客观环境，也要配合主观努力。主观条件只有适应客观环境时，才能发挥作用。
